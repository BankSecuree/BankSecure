Perguntas Básicas
1. Como funciona o monitoramento da Bank Secure?

Primeiramente, nós capturamos os dados de máquina do seu processador, memória e disco. Após isso, os dados são salvos no banco de dados, onde conseguimos emitir mensagens de alerta quando os dados estiverem a cima da média esperada.

2. Meus dados não estão atualizando, o que fazer?

Você pode entrar em contato conosco imediatamente para resolvermos o seu problema.

3. Vocês realizam manutenção dos servidores?

Não. Nossa empresa possui o intuito de proporcionar alertas para que os bancos tomem alguma providência diante disso.
